# CARSI登录教程

TIP

此文档写于 2023.1月,如遇失效请联系我

# 浏览器方式

  1. 登录长江师范学院网上办事大厅

  2. 点击左上角的 可用应用-->信息化服务-->CARSI

# 今日校园方式

  1. 点击服务,点击carsi

TIP

以下两种方法需加入长师信息服务的企业微信, 未加入请联系你的辅导员

# 微信方式

  1. 打开长师信息服务(在微信的通讯录界面)

  2. 选择CARSI应用

# 企业微信方式

  1. 仅需在工作台找到CARSI应用
